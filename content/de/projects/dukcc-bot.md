## Über das Projekt

dukcc-bot ist ein Typescript basierter Discord-Bot, welcher viele proprietäre Features für das PenguinNetwork bereitstellt.  
Darunter zählen
- Automatisches Zuweisen von Rollen basierend auf Spielzeit
- Synchronisierung von Rollen zwischen Game Servern und Discord
- Verknüpfen von Spiel Accounts und Discord Accounts
- Automatisierte Verwaltung von Giveaways basierend auf Spielzeit der Nutzer
- Discord Invite Tracking System um effektivität von Werbung zu beurteilen
- REST Schnittstellen für Integration mit anderen Diensten, wie beispielsweise der Website
