## About the Project

Marketing and property website for a Spanish real estate agent operating on the Costa del Sol.

A Symfony backend was developed to handle data storage and the import of data from an XML feed provided by Spain's largest real estate portal.  
The frontend was built with Nuxt.JS and Tailwind.  

## My Role
I built the project from the ground up as a full-stack developer and was responsible for the entire technical planning and implementation in collaboration with the client.

## Import
The import is triggered asynchronously via RabbitMQ as a queue system, either directly by the provider or manually with a custom XML file.  
The import progress is displayed and updated live in the back office.
