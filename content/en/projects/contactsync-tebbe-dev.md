## About the Project
A tool to sync my contacts from Google Contacts with my IP phone  
  
For a while I used a Grandstream IP phone, which had the ability to import a contact book as XML from a web address.  
What was missing was a way to convert my Google Contacts address book into such an XML file, so I built this project.  
After logging in with their Google account, a unique link is generated that simultaneously contains a password and serves a corresponding XML document.
