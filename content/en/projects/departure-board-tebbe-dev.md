## About the Project

departure-board.tebbe.dev is an interactive web service that displays real-time departure information for public
transit. The service offers a clear user interface, built with Nuxt.js and Vue.js, that fetches current timetable
data via REST APIs.  
Users first specify a stop and then all desired modes of transport, after which a URL is generated that can be used to display the same view at any time.  
→ Ideal for displaying statically on a monitor or embedding in a dashboard.

The tool is aimed at commuters, travelers, and anyone who needs quick information about departure times. The
query happens in real time and delivers detailed information such as departure times, delays, line numbers, and
destinations.
