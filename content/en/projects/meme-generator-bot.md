## About the Project

meme-generator-bot is a Typescript-based Discord bot that lets you generate imgflip.com memes and post them directly into a Discord channel.

The bot provides a slash command that can be used to create a meme and post it in a specified channel.
The tool is also "user installable", which means the bot can also be used in direct messages and on Discord servers where the bot isn't installed.
