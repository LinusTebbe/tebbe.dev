## About the Project
Commissioned by two founders, this involved further developing a web platform that lets end customers find inspection centers for various examinations and book an appointment directly online.  
The central component was search by various criteria, such as postal code, location, examination type, and availability.

To save time and costs, the platform was built on top of an existing project that already provided the basic functionality, though it didn't quite match the founders' wishes.  
Search performance was not optimal, and a radius/location-based search wasn't possible yet either.
This was fixed through several query and index optimizations in the MySQL database.  

The platform was then further developed step by step into a fully functioning platform.  
A back office was built for inspection centers, letting them connect their calendar and manage their own master data.
