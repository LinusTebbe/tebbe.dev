## About the Project

Routing is a Terraform configuration that represents the routing infrastructure of the PenguinNetwork.  
It handles the automatic creation and provisioning of Hetzner Cloud servers.  
These servers are used to actively steer player traffic and thereby avoid unfavorable intercontinental routes.
