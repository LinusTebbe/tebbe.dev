## About the Project

whois.tebbe.dev is a free web service that makes it possible to quickly and easily look up WHOIS information
for IP addresses and domains. The service offers a clear user interface, built with Nuxt.js, that uses
ARIN's API directly as its backend.

The tool is aimed at developers, system administrators, and anyone who needs quick information about IP addresses.
The lookup happens in real time and provides detailed information such as registrar,
registration date, name servers, and other relevant WHOIS data.
